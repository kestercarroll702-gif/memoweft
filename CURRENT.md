# Current Status

MemoWeft is currently focused on library/API clarity, documentation quality, and ecosystem adoption.

- MemoWeft remains library-first.
- The bundled host remains a reference demo.
- Core behavior, cognition rules, and public APIs should not be changed casually.
- Repository cleanup should improve public clarity without changing runtime behavior.
